# Customer-Churn-Prediction-Customer-Lifetime-Value-CLV-Optimization-System

🟡 S — Situation —> “Telecom and e-commerce companies in Germany, such as Deutsche Telekom, face significant revenue loss due to customer churn. However, they often lack a structured way to identify which customers are both likely to churn and highly valuable

🔵 T — Task

My goal is to build a data-driven system to:
	1.	Predict which customers are likely to churn
	2.	Estimate Customer Lifetime Value (CLV)
	3.	Help the business prioritize retention efforts toward high-value customers

🟢 A — Action
an end-to-end pipeline:
1. Data Preparation: Cleaned and processed customer data, handled missing values, and encoded categorical features
2. Feature Engineering: Created variables such as tenure, average spend, and service usage patterns
3. Churn Modeling: Built and compared models including Logistic Regression and Random Forest to predict churn probability
4. CLV Estimation: Calculated customer lifetime value using revenue-based formulas based on tenure and spending
5. Segmentation: Combined churn probability and CLV to classify customers into segments like:
High Value – High Risk
Low Value – High Risk
6. Deployment: Developed an interactive app using Streamlit and deployed it via Streamlit Community Cloud, allowing users to input customer data and receive predictions in real time

🔴 R — Result

The project enabled identification of high-risk, high-value customers, helping simulate a targeted retention strategy.

For example, focusing on the top segment of high-value churn-risk customers can help reduce potential revenue loss and improve customer retention efficiency.

Additionally, the deployed application provided a practical tool for business stakeholders to interact with the model and understand its impact.
